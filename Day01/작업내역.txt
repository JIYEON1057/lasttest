[융합 예술 치료 사이트 - 그림판 및 AI 노래 생성 기능 구현 내역]

=== 최종 구현 현황 ===

1. Day01/001.html
   - <!DOCTYPE html> 문서 구조 정립
   - 캔버스(1000x600) UI 구성: 그림판 영역 설정
   - 도구모음(toolbar) 구성:
     * 색상 버튼 7개 (빨강, 주황, 노랑, 초록, 파랑, 보라, 검정)
     * 커스텀 컬러 선택기(input type="color")
     * 지우개 버튼
     * 지우개 크기 선택 드롭다운 (얇게/보통/굵게/매우 굵게)
     * 전체 지우기 버튼
   - 노래 생성 버튼 및 결과 표시 영역
   - 캔버스를 div#canvas-wrap으로 감싸서 미리보기 오버레이와 캔버스를 정확히 겹침

2. 001.js
   - 캔버스 초기화: 흰색 배경으로 설정
   - 색상 기능:
     * 미리정의된 7가지 색상 버튼 클릭 시 펜 색상 변경
     * 커스텀 컬러 선택기로 자유로운 색상 선택 가능
   - 지우개 기능:
     * 지우개 버튼 클릭 시 지우개 모드 전환
     * 지우개 크기 드롭다운으로 지우는 범위 조절 (10~80px)
     * 원형 미리보기 오버레이(preview canvas)로 지우는 범위를 사전에 표시
     * clearRect()와 clip()을 조합하여 원형 영역을 정확하게 투명하게 지움
   - 전체 지우기:
     * 캔버스 전체를 흰색으로 다시 칠해 초기화
   - 마우스 이벤트:
     * 캔버스에서 마우스 누르면 드로잉 시작
     * 마우스 드래그로 연속적으로 선(펜) 또는 지우개 적용
     * 마우스가 캔버스 밖으로 나갔다가 다시 들어와도 누르고 있으면 드로잉 계속
     * mousemove 이벤트로 부드러운 드로잉 실현
   - 노래 생성 버튼:
     * 클릭 시 캔버스를 base64 PNG로 변환
     * 그림 데이터를 바탕으로 AI가 노래를 생성하는 흐름 구조 (실제 AI API 연동 필요)
     * 2초 후 생성 완료 메시지 표시 (데모)

=== 기술 구현 세부사항 ===
- HTML5 Canvas API 사용
- Canvas 2D Context의 메서드: fillRect, clearRect, clip, stroke, arc 등
- globalCompositeOperation으로 투명한 배경 처리
- 마우스 이벤트: mousedown, mouseup, mousemove, mouseleave, mouseenter
- 오버레이 캔버스를 이용한 미리보기 표시 (pointerEvents: none으로 이벤트 투과)
