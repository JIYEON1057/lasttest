[융합 예술 치료 사이트 - Art2Music 구현 내역]

=== 최종 구현 현황 ===

1. Day01/index.html
   - <!DOCTYPE html> 문서 구조 정립
   - 캔버스(1000x600) UI 구성: 그림판 영역 설정
   - 도구모음(toolbar) 구성:
     * 색상 버튼 7개 (빨강, 주황, 노랑, 초록, 파랑, 보라, 검정)
     * 커스텀 컬러 선택기(input type="color")
     * 지우개 버튼
     * 지우개 크기 선택 드롭다운 (얇게/보통/굵게/매우 굵게)
     * 전체 지우기 버튼
   - 음악 생성 버튼 및 결과 표시 영역
   - Spotify 노래 추천 섹션:
     * Client ID / Client Secret 입력 필드
     * 노래 추천 받기 버튼
     * Spotify Embed 플레이어 표시 영역
   - 캔버스를 div#canvas-wrap으로 감싸서 미리보기 오버레이와 캔버스를 정확히 겹침

2. Day01/main.js
   - 캔버스 초기화: 흰색 배경으로 설정
   - 색상 기능:
     * 미리정의된 7가지 색상 버튼 클릭 시 펜 색상 변경
     * 커스텀 컬러 선택기로 자유로운 색상 선택 가능
   - 지우개 기능:
     * 지우개 버튼 클릭 시 지우개 모드 전환
     * 지우개 크기 드롭다운으로 지우는 범위 조절 (10~80px)
     * 원형 미리보기 오버레이(preview canvas)로 지우는 범위를 사전에 표시
     * clearRect()와 clip()을 조합하여 원형 영역을 정확하게 투명하게 지움
   - 전체 지우기:
     * 캔버스 전체를 흰색으로 다시 칠해 초기화
   - 마우스 이벤트:
     * 캔버스에서 마우스 누르면 드로잉 시작
     * 마우스 드래그로 연속적으로 선(펜) 또는 지우개 적용
     * 마우스가 캔버스 밖으로 나갔다가 다시 들어와도 누르고 있으면 드로잉 계속
     * mousemove 이벤트로 부드러운 드로잉 실현
   
   - Web Audio API 음악 생성 기능 (무료):
     * 캔버스 색상 분석 (빨강, 주황, 노랑, 초록, 파랑, 보라 비율)
     * 색상별 무드 판단 (밝은/어두운/차분한/에너지틱한)
     * 색상 기반 음계 및 코드 결정
     * 오실레이터(sine, triangle, sawtooth)로 멜로디 생성
     * 패드 사운드로 배경음 생성
     * 10초간 자동 생성 음악 재생
     * 정지 버튼으로 음악 중단 가능
   
   - Spotify 노래 추천 기능:
     * Client Credentials Flow로 Spotify API 인증
     * 캔버스 색상 분석 → 무드 키워드 추출
     * Spotify Search API로 무드에 맞는 노래 검색
     * Spotify Embed 플레이어로 페이지 내에서 직접 노래 재생 가능
     * 최대 3곡 추천 표시

3. Day01/style.css
   - 다크 테마 UI 디자인
   - 캔버스 및 도구모음 스타일링
   - 색상 버튼 hover 효과 및 active 상태 표시
   - 음악 생성 결과 영역 스타일
   - Spotify 섹션 스타일:
     * API 키 입력 필드 디자인
     * Spotify 브랜드 컬러(#1DB954) 적용
     * Embed 플레이어 컨테이너 스타일

=== 기술 구현 세부사항 ===
- HTML5 Canvas API 사용
- Canvas 2D Context의 메서드: fillRect, clearRect, clip, stroke, arc 등
- globalCompositeOperation으로 투명한 배경 처리
- 마우스 이벤트: mousedown, mouseup, mousemove, mouseleave, mouseenter
- 오버레이 캔버스를 이용한 미리보기 표시 (pointerEvents: none으로 이벤트 투과)
- Web Audio API: OscillatorNode, GainNode, AudioContext
- Spotify Web API: Client Credentials Flow, Search Endpoint
- Spotify Embed Player: iframe으로 페이지 내 음악 재생

=== 사용 방법 ===
1. 캔버스에 그림 그리기
2. "음악 생성하기" 버튼 → Web Audio로 즉석 음악 생성
3. Spotify 추천:
   - Spotify Developer에서 앱 생성 (https://developer.spotify.com)
   - Client ID와 Client Secret 입력
   - "노래 추천 받기" 버튼 클릭
   - Embed 플레이어에서 바로 노래 감상
